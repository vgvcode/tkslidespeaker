Steps to get Windows machine ready to run SlideSpeaker

Install Python for windows
https://www.python.org/downloads/windows/

Install Chrome for Windows
//www.google.com/chrome/next-steps.html?installdataindex=empty&statcb=1&defaultbrowser=0&brand=IMSH

Install Pip
curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py

Install aws cli

Install vs code
https://code.visualstudio.com/download

Go to the tkslidespeaker directory
pip install -r requirements.txt
(set pillow version <=9.5.0)
